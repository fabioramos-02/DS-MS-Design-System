# Parágrafos — uso e acessibilidade

Baseado em [designsystem.ms.gov.br/paragrafos](https://www.designsystem.ms.gov.br/paragrafos/).
Classe CSS + `<p>` nativo — nunca `<div>` ou `<br>` pra espaçamento.

## Variantes

| Variante | Uso | Token de cor |
|---|---|---|
| Padrão | Conteúdo institucional, descrições | `--paragrafos-padrao-color` (neutral-900) |
| Secundário | Metadados, notas complementares | `--paragrafos-secundario-color` (neutral-500) |
| Destaque | Ênfase pontual, sem substituir títulos | `--paragrafos-destaque-color` + peso 600 |

## Regras tipográficas

- `line-height`: `--paragrafos-line-height` (24px sobre 16px de fonte ≈ 1.5).
- Largura máxima `--paragrafos-max-width` (70ch) — alinhado à faixa de 60–80 caracteres por linha.
- Espaçamento entre parágrafos via `--paragrafos-margin-bottom`, nunca `<br><br>`.

## Acessibilidade (eMAG + WCAG 2.1 AA)

| Categoria | O que verificar | Como |
|---|---|---|
| **Contraste** | ≥ 4.5:1 mesmo na variante secundária | `neutral-500` sobre fundo branco passa AA pra texto ≥ 14px |
| **Zoom** | Legível e sem corte a 200% | `max-width` em `ch` escala com a fonte |
| **Leitor de tela** | `<p>` é anunciado como bloco de texto corrido | NVDA / VoiceOver |
