# Link, Card e Tag — uso e acessibilidade

## Link
- Cor `--color-info-600` (não usa `--color-primary-500` para não confundir com botão).
- Sublinha no hover — diferença de **estilo**, não só cor, ajuda quem não distingue cores.
- `:focus-visible` com outline — igual ao padrão dos demais componentes.

## Card
- `.card` é um contêiner neutro; `.card-link` é o card inteiro clicável (`<a>`), com foco visível no elemento todo.
- Use heading semântico (`<h3>`, não `<span>`) para o título se o card representar uma seção real da página — aqui usamos `<span>` só por ser dentro de um link.

## Tag
- Tags de feedback (sucesso/erro/alerta/info) são **decorativas + texto** — nunca só cor. O rótulo de texto já diz "Erro", "Sucesso" etc., então a informação não depende de percepção de cor.

## Código

```html
<a class="link" href="#">Saiba mais</a>

<a class="card-link" href="#">
  <span class="card-link-title">Título</span>
  <span class="card-link-body">Descrição</span>
</a>

<span class="tag tag-error">Erro</span>
```
