# Notification — uso e acessibilidade

Banners inline de feedback. Quatro variantes: padrão (info), sucesso, erro, alerta.

## Acessibilidade
- `role="status"` para mensagens informativas (sucesso/info) — leitor de tela anuncia sem interromper.
- `role="alert"` para erro/alerta — leitor de tela anuncia **imediatamente**, mesmo sem foco no elemento.
- Borda lateral de 4px + ícone (quando usado) reforçam o significado sem depender só da cor de fundo.

## Código

```html
<div class="notification is-error" role="alert">Erro: não foi possível enviar.</div>
```
