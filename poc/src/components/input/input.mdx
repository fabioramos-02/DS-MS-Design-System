# Input — uso e acessibilidade

Campo de texto, textarea e select compartilham a classe base `.field` + variante (`.field-input`, `.field-textarea`, `.field-select`).

## Quando usar
- **Texto** — entradas curtas de uma linha.
- **Textarea** — entradas longas (descrições, observações).
- **Select** — escolha entre opções fechadas e conhecidas.

## Acessibilidade (eMAG + WCAG 2.1 AA)

| Categoria | O que verificar |
|---|---|
| **Rótulo** | `<label for>` sempre associado ao `id` do campo — leitor de tela anuncia o nome do campo. |
| **Erro** | `.is-error` muda a borda E o texto de ajuda (`.field-helper.is-error`) — nunca só a cor (não depender de cor isoladamente). |
| **Foco** | Borda azul + halo (`box-shadow`) — visível sem depender de cor isolada. |
| **Desabilitado** | Atributo `disabled` nativo — leitor de tela anuncia o estado. |

## Código

```html
<div class="field">
  <label class="field-label" for="nome">Nome completo</label>
  <input class="field-input" id="nome" type="text" placeholder="Digite seu nome" />
  <span class="field-helper">Como aparece no seu documento.</span>
</div>
```
