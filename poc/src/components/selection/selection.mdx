# Checkbox, Radio e Toggle — uso e acessibilidade

Controles de seleção nativos (`<input type="checkbox">`, `type="radio"`, e toggle = checkbox com `role="switch"`), estilizados via `appearance: none`.

## Acessibilidade
- Sempre dentro de `<label>` (ou com `aria-label`) — clique na label também marca o controle.
- `role="switch"` no toggle — leitor de tela anuncia "ativado/desativado" em vez de "marcado/desmarcado".
- `:focus-visible` com outline 2px (`--color-info-600`) — igual ao padrão do Button.
- Não depende só de cor: checkbox marcado tem ícone de check; radio marcado tem ponto interno; toggle marcado move a bolinha — diferença de **forma**, não só cor.

## Código

```html
<label><input class="checkbox" type="checkbox" /> Aceito os termos</label>
<label><input class="radio" type="radio" name="opcao" /> Opção A</label>
<input class="toggle" type="checkbox" role="switch" aria-label="Notificações" />
```
