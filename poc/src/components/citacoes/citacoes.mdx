# Citações — uso e acessibilidade

Baseado em [designsystem.ms.gov.br/citacoes](https://www.designsystem.ms.gov.br/citacoes/).
`<blockquote>` pra citação longa + `<cite>` pra atribuição — nunca aspas soltas num `<p>`.

## Estrutura

- Barra lateral (`border-left`, `--citacoes-border-color`) é **recurso complementar**, não o único
  diferenciador — reforçada por `font-style: italic` no texto.
- Ideal 1–4 linhas, mantendo objetividade.
- Margens claras (`--citacoes-padding`) evitam fusão visual com parágrafos vizinhos.

## Tokens

| Token | Valor |
|---|---|
| `--citacoes-border-color` | `primary-600` |
| `--citacoes-background` | `primary-100` |
| `--citacoes-font-size` / `--citacoes-line-height` | 18px / 28px |
| `--citacoes-cite-color` | `neutral-500` |

## Acessibilidade (eMAG + WCAG 2.1 AA)

| Categoria | O que verificar | Como |
|---|---|---|
| **Contraste** | Texto e atribuição ≥ 4.5:1 | `neutral-900`/`neutral-500` sobre `primary-100` passam AA |
| **Semântica** | `<blockquote>`/`<cite>` anunciados como citação | NVDA / VoiceOver |
| **Sem cor isolada** | Barra lateral não é o único sinal — itálico reforça | Revisão visual com escala de cinza |

Sempre preservar o texto original em citações diretas; nunca usar `<blockquote>` só por efeito visual.
